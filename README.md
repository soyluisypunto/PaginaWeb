## Descripción del proyecto
Este proyecto es una plataforma web educativa desarrollada con el objetivo de introducir a los usuarios al mundo de la música y los instrumentos musicales. La página cuenta con una portada, una sección principal con información básica sobre distintos instrumentos, una encuesta donde los usuarios pueden dejar comentarios y sugerencias, y un foro donde estos comentarios se muestran dinámicamente. La aplicación integra frontend, backend y base de datos para ofrecer una experiencia completa e interactiva.

## Lenguajes y tecnologías utilizadas
- **HTML** – Estructura de las páginas web  
- **CSS** – Estilos y diseño visual  
- **JavaScript** – Interactividad en el navegador (mensajes, botones, eliminación de comentarios)  
- **Python (Flask)** – Backend y manejo de rutas  
- **SQL (PostgreSQL)** – Almacenamiento de datos  
- **Git & GitHub** – Control de versiones y hospedaje del proyecto
- **Render** – Hospedaje de la aplicación y base de datos

## Base de datos
La aplicación utiliza una base de datos **PostgreSQL** alojada en **Render**, donde se almacenan los datos enviados por los usuarios a través del formulario de encuesta, incluyendo nombre, correo electrónico, comentarios y fecha de envío.
